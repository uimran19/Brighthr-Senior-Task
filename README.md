# BrightHR Senior Tech Task

## Setup & Installation

1. clone the repo

`git clone https://github.com/uimran19/Brighthr-Senior-Task.git`

2. cd into the folder and install dependencies

```bash
cd brighthr-senior-task
npm install
```

## Development

`npm run dev`

## Testing

For unit testing, React Testing Library and Jest was used.

To run the tests, input the following command:

`npm run test`

## Description

An absences app, where users can view current employee absences and sort by name, date and absence type.
